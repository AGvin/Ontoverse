Ontoverse може бути цікавим як приклад побудови pre-formal framework: як неінституційний автор організовує концептуальні інтуїції, аналогії, термінологію та спекулятивні структури до математичної формалізації або відкидання.

## Переклади

- [English](../../)
- Українська — поточна

Проєкт не подається як доказ. Він подається як матеріал для перевірки, критики, порівняння та можливої формалізації.

## Чим є Ontoverse

Ontoverse є самостійно розробленим концептуальним фреймворком для організації реальності, понять і можливих прихованих звʼязків між ними.

Він структурований як research notebook, що містить definitions, conceptual models, diagrams, hypotheses, analogies, open problems і comparisons with existing scientific and philosophical concepts.

## Чим Ontoverse не є

Ontoverse наразі не заявляється як:

- завершена фізична теорія;
- заміна квантовій механіці або теорії відносності;
- доказ hidden variables;
- доказ many-worlds або іншої інтерпретації;
- новий експериментально підтверджений закон;
- містичне або одкровенне твердження.

## Корисна критична позиція

Ontoverse варто розглядати з конструктивним скепсисом. Мета не в тому, щоб захищати кожен термін, а в тому, щоб відокремити useful abstractions, misleading analogies, terminology that should be renamed, unresolved parts і parts that should be rejected.
