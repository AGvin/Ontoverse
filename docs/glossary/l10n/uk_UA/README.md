Визначення можуть змінюватися в міру уточнення фреймворку.

Якщо явно не зазначено інше, Ontoverse-specific terms є proposed conceptual terms, а не established scientific terminology. Їх слід читати як working definitions для pre-formal framework.

## Ключові терміни

- **History-space** (простір історій) — structured space of possible histories у фреймворку Ontoverse.
- **Frontal time** (фронтальний час) — запропонований global ordering parameter для розгортання history-space.
- **Frontal time plane** (площина фронтального часу) — рубінова visual boundary, що представляє specific frontal-time slice.
- **Local time** (локальний час) — час, досвідчений уздовж particular historical trajectory.
- **Temporal density** (темпоральна щільність) — запропонована density of significant event-nodes per unit of frontal time.
- **Event-node** (подія-вузол) — significant transition point у historical trajectory.
- **Compatibility channel** (канал сумісності) — access structure, у якій можуть взаємодіяти лише mutually compatible states, records та observers.
- **Convergent channel** (конвергентний канал) — compatibility channel, у який входить більше ніж одна distinct historical trajectory.
- **Causal processing density** (щільність причинної обробки) — запропонований Ontoverse term для amount of local state change, interaction та causal coordination у регіоні relative to frontal time.
