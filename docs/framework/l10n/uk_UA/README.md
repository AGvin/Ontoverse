## Мінімальне твердження

Мінімальне твердження полягає в тому, що досвідчену реальність може бути корисно описувати як шлях крізь структурований простір можливих історій.

У цьому формулюванні:

- історії не є лише лінійними timeline;
- історії можуть розходитися в несумісні продовження;
- історії можуть мати спільні локально еквівалентні future channels;
- спостерігачі мають доступ лише до світів, сумісних з їхніми локальними записами та взаємодіями;
- локально досвідчений час може бути повʼязаний зі щільністю значущих event-nodes уздовж історії;
- frontal time може розглядатися як глобальний параметр упорядкування для розгортання history-space.

## Межі

Ontoverse не стверджує, що замінює квантову механіку, розвʼязує проблему вимірювання або доводить новий закон фізики.

Сторінки поруч із фізикою мають явно розділяти established background, interpretational analogy, speculative model components та open formalization tasks.
