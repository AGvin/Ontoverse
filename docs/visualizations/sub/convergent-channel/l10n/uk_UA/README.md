Ця діаграма показує випадок, коли кілька відмінних історичних траєкторій поступово входять у спільну локально сумісну future-access structure біля frontal time plane.

## Переклади

- [English](../../)
- Українська — поточна

## Що показує діаграма

Діаграма має три концептуальні області:

- **distinct origins** — траєкторії починаються з різних історичних витоків;
- **divergence** — траєкторії розходяться та проходять через різні event-nodes;
- **compatibility filtering** — лише частина структур лишається локально сумісною з майбутньою спільною областю;
- **convergent channel** — траєкторії сходяться в один локально еквівалентний канал біля frontal time plane.

Усі видимі траєкторії завершуються на frontal time plane.

## Інтерпретація

Convergent channel не означає, що минуле стає однаковим.

Він означає, що різні історичні походження можуть входити в спільний локально сумісний майбутній доступний стан або канал, якщо їхні відмінності більше не мають локально релевантного впливу для показаного model slice.

## Важливе обмеження

Діаграма не показує реалізовані події за frontal time plane.

Фронтальна площина представляє boundary of present у цій візуалізації. Вміст за нею належав би future-side structure, яку поточна сторінка моделі ще не визначає.

## Роль у документації

Використовуйте цю візуалізацію для пояснення:

- compatibility channels;
- convergence;
- локальної еквівалентності після різного походження;
- фільтрації історій через compatibility constraints.
