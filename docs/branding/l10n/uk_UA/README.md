## Основний логотип

Основний логотип — це темний неоновий wordmark Ontoverse з емблемою history-space та прямокутною рубіновою frontal time plane.

## Правило asset

Не замінюйте затверджений візуальний напрям спрощеними SVG approximation, якщо автор явно не затвердив саме таку заміну.

Деталізовані SVG assets є бажаними для документації репозиторію, бо вони зберігаються в source control, придатні до review та редагування. Raster export може бути додано пізніше для контекстів, де PNG або WebP rendering виглядає краще.

Якщо raster export буде додано, зберігайте SVG як editable source, якщо автор явно не позначить raster image як visual source of truth.
