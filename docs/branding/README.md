# Branding

Status: draft

Ontoverse branding uses the author-approved dark neon conceptual physics style.

## Primary Logo

The primary logo is the dark neon Ontoverse wordmark with a history-space emblem and a rectangular ruby frontal time plane.

Expected asset path:

```text
assets/images/ontoverse-logo-frontal-plane-rectangular.png
```

## Alternative Logo

The alternative logo uses the same visual language with an oval or membrane-like ruby frontal time plane.

Expected asset path:

```text
assets/images/ontoverse-logo-frontal-plane-oval.png
```

## Asset Rule

Do not replace the approved raster-style artwork with simplified SVG approximations unless the author explicitly approves that exact replacement.

If an editable SVG is added, keep it as an alternative or source approximation and clearly document whether the raster image or SVG is the visual source of truth.
