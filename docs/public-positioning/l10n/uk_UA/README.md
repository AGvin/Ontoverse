# Публічне позиціонування

Статус: draft

Ontoverse є самостійно розробленим концептуальним фреймворком для організації реальності, понять і можливих прихованих зв’язків між ними.

Його не слід представляти як завершену наукову теорію, доказ, містичне твердження або заміну усталеній фізиці.

Публічна цінність Ontoverse є дослідницькою: він зберігає спекулятивні інсайти у структурованій формі, щоб їх можна було перевіряти, критикувати, порівнювати з наявними ідеями, уточнювати, формалізувати або відкидати.

## Запропонований однорядковий опис

Самостійно розроблений концептуальний фреймворк для організації реальності, понять і можливих прихованих зв’язків між ними.

## Зручне для медіа питання

Чи може особистий концептуальний фреймворк виявити корисну структуру до того, як наука матиме для неї формальну назву?

## Обіцянка читачеві

Ontoverse має чітко показувати, що стверджується, що лише досліджується, а що залишається невирішеним.

Скептичний читач повинен мати змогу визначити:

- основну інтуїцію;
- запропоновану термінологію;
- аналогію або мотиваційну ідею;
- зв’язок із наявними концепціями;
- припущення;
- слабкі місця;
- відкриті завдання формалізації.

## Варто використовувати

Використовуйте такі формулювання:

- концептуальний фреймворк;
- доформальна модель;
- структурований дослідницький блокнот;
- спекулятивна, але обмежена гіпотеза;
- інтерпретаційна аналогія;
- запропонована термінологія;
- відкрите завдання формалізації;
- порівняння з наявними концепціями;
- раціональне зерно для майбутнього відкриття або підтвердження.

## Слід уникати

Уникайте формулювань, які натякають на:

- доказ;
- остаточну істину;
- містичний авторитет;
- походження зі сну, якщо автор явно цього не задокументував;
- заміну усталеної фізики;
- розв’язання квантової механіки;
- новий закон фізики;
- усталене наукове підтвердження;
- приховану істину, розкриту без формального обґрунтування.

## Безпечний публічний підсумок

Ontoverse не стверджує, що нову фізичну теорію було доведено. Це спроба організувати набір концептуальних інтуїцій, гіпотез та інтерпретаційних моделей так, щоб їх можна було перевіряти, оскаржувати, порівнювати, уточнювати, формалізувати або відкидати.

Проєкт має цінність лише тією мірою, якою його структура допомагає виявити, які частини корисні, які дублюють наявні ідеї, які вводять в оману, а які можуть заслуговувати на подальшу формалізацію.
